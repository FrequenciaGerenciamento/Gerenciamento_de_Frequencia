<h1 align="center"> Gerenciamento_de_Frequencia </h1

 <p align="justify"> Este gerenciamento de frequência, tem o objetivo de monitorar o tempo que cada funcionario dedica ao trabalho. Ele permite registrar entradas e saídas, além de oferecer funcionalidades para cadastrar, buscar, atualizar e excluir dados relacionados à frequência dos funcionários.</p>

* APPS 

- users - É responsável por armazenar tudo relacionado ao gerenciamento de usuários, como cadastro, consulta, atualização e exclusão de dados.

- frequencias - É responsavel por monitorar o tempo de trabalho de cada funcionário
